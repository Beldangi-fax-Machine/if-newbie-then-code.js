ü§î What is Code Structure?

Code structure is like writing a recipe! Just like you follow steps to bake cookies, computers follow steps in your code. Let's learn how to organize these steps!

1 Variables: These are like boxes where you store information
let playerName = "Alex"; // A box holding the player's name let score = 0; // A box holding the score
2 Functions: These are like special machines that do tasks
function sayHello(name) { return "Hello, " + name + "! üëã"; }
3 If Statements: These help computers make decisions
if (score > 10) { // Do this if score is greater than 10 console.log("You're doing great!"); } else { // Do this if score is 10 or less console.log("Keep trying!"); }
4 Loops: These repeat actions multiple times
for (let i = 1; i <= 3; i++) { console.log("Jump number " + i); // Jump 3 times! }

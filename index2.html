üéí What are Arrays?

Arrays are like backpacks that can hold multiple items! Instead of having one variable for each item, you can store many items in one array.

1 Creating Arrays: Store multiple values in one place
let fruits = ["apple", "banana", "orange"]; // An array with 3 fruits let scores = [95, 87, 92, 88]; // An array with 4 scores console.log(fruits[0]); // Gets "apple" (arrays start at 0!) console.log(fruits[1]); // Gets "banana"
2 Array Methods: Cool things you can do with arrays
let colors = ["red", "blue"]; colors.push("green"); // Add to end: ["red", "blue", "green"] colors.pop(); // Remove from end: ["red", "blue"] console.log(colors.length); // Shows 2 (how many items)
3 Looping Through Arrays: Visit each item in the array
let pets = ["dog", "cat", "bird"]; for (let i = 0; i < pets.length; i++) { console.log("I have a " + pets[i]); } // Even easier way! pets.forEach(function(pet) { console.log("My " + pet + " is cute!"); });

üì¶ What are Objects?

Objects are like labeled boxes! Each item has a name (key) and a value. Think of it like a character in a video game with different stats.

4 Creating Objects: Store related information together
let player = { name: "Alex", level: 5, health: 100, isAlive: true }; console.log(player.name); // Shows "Alex" console.log(player.level); // Shows 5 player.health = 95; // Change health to 95
5 Objects with Methods: Objects can have actions too!
let dog = { name: "Buddy", age: 3, bark: function() { console.log("Woof! My name is " + this.name); } }; dog.bark(); // Calls the bark function

üí° Key Takeaways!

1. Arrays store multiple items in order - access them with [0], [1], [2], etc.

2. Objects store related information with labels (keys and values)

3. Use .push() to add items to arrays, .pop() to remove from the end

4. Loop through arrays to work with each item

5. Arrays can hold objects, and objects can hold arrays! ü§Ø
